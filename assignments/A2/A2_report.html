Subtheme: Corporate Landlords & Evictions

Overall Analysis Questions

  1. Is there a relationship between the rate of corporate landlord and residential landlord throughout the years in Boston's neighborhoods?
  2. Which neighborhoods have had the biggest change from resident landlord to corporate landlord in the last 20 years?
  3. Which communities live in the neighborhoods where the biggest change from resident landlord to corporate landlord in the last 20 years occured?

Reasons to choose this subtheme

In the readings and the page of the class that talks about corporate landlords and evictions, we can see that it mentions multiple times how communities of color, especially Black and Latinx people, are the ones that face a higher eviction rate. In this assignment, I will use the data provided by the course to show which neighborhoods of Boston are most affected by corporate landlord compared to resident landlord and which communities are part of those neighborhoods. I will also be looking at which neighborhoods have changed the landlords from residential to corporate in the last 20 years and look as well at which communities are part of those neighborhoods. All the questions were made to see, with the visualizations shown below, how much the Black and Latinx communities are affected.

Discoveries & Insights

First, note that the file Corp_Ownership_and_Occupancy_Over_Time.csv is a partition of the first four columns of Census_and_Corp_Ownership_and_Occupancy_Over_Time.csv. The file that has all the data (the one that was partitioned), from this point forward, will be referred as the complete file. For this project, we only need the complete file and the file we are given with the zip codes of the Boston neighborhoods (Boston_Neighborhoods_Zipcodes.csv) to create the map in Tableau. I couldn't have two different datasets in one, so I had to merge the zip code file with the complete file, using pandas. Some neighborhoods in the complete file do not appear in the zip codes file making the map not complete in figure 4. The order of the columns in the complete file do not make sense for a reader, but we do not have to reorder them for this project since we are using Tableau to create visualizations and the order of the data does not matter.

Important to note that the first two visualization do not use percentage and the other visualizations do use percentage.
